```python
# Instala las librerias
pip install -r requirements.txt

# Ejecuta
python3 alert.py <name_software>

# Ejemplo
python3 alert.py DeskTime

# ctrl + c para cancelar
# el <name_software> debe ser igual a como se muestra en el System Monitor
```
